## 📝 Descrição do Projeto
Este projeto é uma aplicação de gerenciamento de teclados mecânicos. Originalmente concebido para rodar com um backend NestJS, o projeto foi evoluído para utilizar **LocalStorage**, permitindo uma experiência de usuário fluida, offline e com persistência de dados diretamente no navegador.

Ele faz parte de um ecossistema integrado com uma Landing Page de marketing, onde as escolhas feitas no catálogo são sincronizadas automaticamente com esta lista.

## ✨ Funcionalidades Principais
* **Persistência Local**: Os dados não se perdem ao atualizar a página (LocalStorage API).
* **Gestão de Itens**: Adicione à lista de desejos, marque como adquirido ou remova com histórico.
* **Interface Dark Mode**: Design baseado no meu portfólio pessoal com estilo Glassmorphism.
* **Animações Customizadas**: Feedback visual em cada ação do usuário.
